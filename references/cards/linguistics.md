# 语言学方法卡（linguistics）

> 招式定位：描述 / 比较 / 因果 / 解释机制 / 诠释理解。学科轴：语言学（新增轴）。
> 与 `cards/experiment-design.md`（实验范式）、`cards/qualitative.md`（访谈/变异）、`cards/computational.md`（形式化/语料）互补。

## 适用场景
- 句法/语义/语用/音系/语音的**实证检验**（不只内省）。
- 语言加工（心理语言学）、语言变异（社会语言学）、语言使用（语料/计算语言学）。
- 跨语言比较、语言习得、bilingualism。

## 常用方法
- **实验句法/语义**：可接受性判断（Likert/magnitude estimation/强制选择）、自定步速阅读、眼动、ERP、fMRI → 测"结构合不合/加工难度"。详见 `sources/B9-experimental-syntax.md`。
- **语音/音系**：声学分析（Praat）、发音生理（EMA/超声）、感知实验（ABX/辨别）。
- **心理语言学**：自定步速阅读、眼动、ERP、fMRI 看加工时程。
- **社会语言学**：变异分析（GoldVarb）、访谈/民族志看语言与社会变量。
- **语料/计算语言学**：树库/大型语料（频率、搭配、共现）、NLP 模型拟合（见 `cards/computational.md`）。

## 前提
- 被试语言背景须控制（母语者/熟练度）；材料最小配对。
- 实验范式需随机、控制、事前功效（见 playbook 功效节）。

## 常见坑
- **可接受性 ≠ 合语法性**：加工负荷/频率/语用混淆"合语法"判断（见 B9）。
- **内省主导**：只靠单人直觉，不可复现——用实证范式补。
- **语料频率误读**：频次高≠"更合语法"，需控结构干频/语域。
- **跨语言泛化过度**：某结构在一语言模式，不自动成立他语言。
