# 数据分析卡（data-analysis）

> 招式定位：描述 / 比较 / 关联 / 预测。学科轴：通用（所有定量数据）。
> 老衲的"秒定位"心法：不背清单，沿三岔口走——目的 → 数据 → 变量。

## 适用场景
- 已有数据，需选统计方法做描述、组间比较、变量关联、或预测。

## 操作步骤（决策树）
1. **目的**？描述（均值/分布）→ 比较（差异）→ 关联（关系）→ 预测（建模）。
2. **数据类型**？连续 / 分类 / 计数 / 有序 / 文本。
3. **变量结构**？几组、几个自变量、配对/独立、嵌套/重复测量。
4. **验前提**：正态（Shapiro/QQ）、方差齐（Levene）、独立、线性。
5. **出结果**：报**效应量 + 置信区间**，不只 p 值。
6. **校正**：多指标 → Bonferroni/FDR；多重比较必做。
7. **敏感性**：去离群值结果稳不稳？做一遍。

## 速查（常见落点）
- 两组差异（独立/配对）→ t 检验 / 配对 t。
- 三组+差异 → ANOVA（重复测量用重复测量 ANOVA）。
- 关联 → Pearson（连续）/ Spearman（有序）/ 卡方（分类）。
- 谁影响谁 → 回归族（线性/Logistic/多水平，按因变量类型）。
- 预测 → 回归 / 机器学习（需独立验证集）。

## 前提假设
- 各检验的假设（正态/独立/方差齐/样本量）；违则换非参（Mann-Whitney / Kruskal-Wallis）或稳健法。

## 常见坑（破绽速查）
- **相关当因果**：相关只说共变，因果要实验或严谨识别。
- **只看 p 不看效应量**：p<0.05 但效应微小，可能无实际意义。
- **多重比较未校正**：测 20 个指标必有几个假显著。
- **p-hacking**：换指标/分组直到显著，制造假阳性。
- **离群值乱删**：先查是录错还是真信号，做敏感性分析。
- **低功效高估**：样本小易得夸张效应，事前算功效。
- **伪重复**：同一主体多次测量当独立样本，膨胀自由度。

## 何时换招
- 要因果识别 → `experiment-design` 或 `econometrics`。
- 要测抽象构念 → `survey-scale`（先信效度）。
- 要意义/机制 → `qualitative`。

## 经典出处（此招出自何典）
- 效应量/功效根：Cohen《Statistical Power Analysis for the Behavioral Sciences》（书单 #23）。
- 现代贝叶斯教学：McElreath《Statistical Rethinking》（书单 #25）。
- 应用统计入门：Field《Discovering Statistics Using...》（书单 #22）。
- 假阳性必然：Ioannidis 2005（书单 #73）。
- 脉络：p 值由 Fisher 引入、经 Neyman-Pearson 体系化，二者"p 值之争"至今未休；Cohen 力倡"效应量 > p 值"，老衲开口必报效应量+CI。
