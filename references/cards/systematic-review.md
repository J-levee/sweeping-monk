# 系统综述 / Meta 分析卡（systematic-review）

> 招式定位：描述 / 比较 / 批判评价 / 关联。学科轴：全学科（医学/社科/公卫最常用）。
> 与 `research-playbook.md` §六（系统综述 prose）互补：本卡给"取卡即用的操作清单"。

## 适用场景
- 要"这个领域目前知道什么、证据多强"——而非单篇研究。
- 指南/政策/元分析前置；回答"X 干预/暴露的效果有多大、稳不稳"。

## 操作步骤
1. **定 PICO / 研究问题**：明确人群、干预、对照、结局；注册协议（PROSPERO / OSF）。
2. **检索策略**：多库（≥2）+ 多词 + 滚雪球（引文回溯+前向）；留检索式供复现。**只搜一个库是破绽**。
3. **PRISMA 流程**：去重→题摘初筛→全文筛→纳入；画流程图与漏斗图。
4. **偏倚风险**：RCT 用 RoB 2；观察性用 ROBINS-I / Newcastle-Ottawa；逐项评，不略。
5. **效应量合并**：标准化效应量（Cohen's d / OR / RR / HR），报**置信区间**，不只报 p。
6. **异质性**：I² 高（如 >50%）说明研究间差异大——先做亚组/元回归找来源，别硬合并；否则改叙述性合成。
7. **发表偏倚**：漏斗图 + Egger/Begg 检验；若明显，用剪补法或报告"证据可能高估"。
8. **证据分级**：GRADE（随机误差/偏倚/不一致/不直接/发表偏倚 → 高/中/低/极低）。

## 前提假设
- 纳入研究同质（至少临床/方法学可比）；否则合并无意义。
- 检索够全——漏检阳性/阴性研究都会偏。

## 常见坑
- **只报 p**：不报效应量与 CI，无法判"多大/稳不稳"。
- **忽略异质性**：I² 高还强行合并出一个"总效应"，误导。
- **发表偏倚无视**：只纳入已发表（偏阳性）研究，效应高估。
- **选择性结局**：只合成显著结局，略非显著——p-hacking 的综述版。
- **检索不全**：漏关键库/词，结论根基虚。
